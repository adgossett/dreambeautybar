Configuring the contact form

BARBER - A MULTIPURPOSE TEMPLATE Comes with ajax contact form in every html layout.You will find a file named as "ajaxmail.php". In that file you need to add the default email ID's the "Sender" & the "Receiver". (Use any text editor like notepad++)
